Bienvenidos

Pagina web basica creada para el trabajo práctico.
Aca aprenderemos a usar etiquetas HTML esenciales.

Pasos para crear una pagina web:

  1. Escribir el codigo HTML.
  2. guardar el archivo con extension .html.
  3. abrirlo en un navegador web.

Herramientas útiles:

  • Editor de texto (VS Code, Sublime Text, Notepad++).
  • Navegador web (Chrome, Firefox).
  • Conocimientos basicos de HTML y CSS.
