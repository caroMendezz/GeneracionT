Pagina de ejemplo

Se mostrara el siguiente parrafo que tiene estilos aplicados con 7 propiedades CSS diferentes usando el atributo style. Algunas de las propiedades nuevas que se aplican aca son:

  • border-radius: bordes redondeados.
  • box-shadow: sombra alrededor del elemento.
  • max-width: ancho maximo para mejorar la legibilidad.
  • padding: espacio interior para separar el texto de los bordes.
  • background-color: color de fondo suave.
  • color: color del texto (en el h1).
  • text-align: alinear texto al centro.
